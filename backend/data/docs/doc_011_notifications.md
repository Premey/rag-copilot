# Notification & Alert Settings

CloudDesk gives you full control over what notifications you receive and how.

## Notification Channels
- **In-app:** Bell icon in top-right corner. Shows last 50 notifications.
- **Email:** Configure in Settings → Notifications → Email. Choose digest (instant / hourly / daily).
- **Slack:** See Slack Integration guide. Per-event toggles available.
- **Mobile Push:** Download the CloudDesk mobile app (iOS / Android) and enable push.

## Configuring Notifications
1. Settings → Notifications.
2. Toggle categories: Ticket updates, Mentions, SLA alerts, Billing, Team changes.
3. Set per-channel preferences (some alerts can be email-only, others app-only).
4. Click Save.

## Do Not Disturb
Enable DND under Settings → Notifications → Do Not Disturb. Set hours (e.g., 9pm–8am). Critical SLA breach alerts override DND.

## Unsubscribing from a Ticket
Open the ticket → click the bell icon → Unsubscribe. You will no longer receive updates for that ticket.
